Geoservicios OGC
🖼️
Mapas en Web

Servicio WMS — Web Map Service

Visualizá capas geográficas como imágenes generadas dinámicamente: vectoriales, ortofotografías, imágenes satelitales. Compatible con QGIS, ArcGIS y otros SIG.

https://geoportalqa.lujandecuyo.gob.ar/geoserver/wms
📍
Vectores en Web

Servicio WFS — Web Feature Service

Accedé y descargá los datos vectoriales en su formato original. Permite consultar, filtrar y editar features geográficas. Ideal para análisis en SIG de escritorio.

https://geoportalqa.lujandecuyo.gob.ar/geoserver/wfs
🌡️
Coberturas

Servicio WCS — Web Coverage Service

Accedé a datos raster como modelos de elevación, imágenes de satélite y series temporales en formatos interoperables. Orientado a modelado y análisis espacial.
